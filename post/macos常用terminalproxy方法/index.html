MacOS常用TerminalProxy方法

Fri, Aug 14, 2020

首先配置好自己本地的sock监听地址

打开.zshrc

vi ~/.zshrc

增加 proxy | unproxy

alias proxy="
    export HTTP_PROXY=socks://127.0.0.1:1080;
    export HTTPS_PROXY=socks://127.0.0.1:1080;
    export ALL_PROXY=socks://127.0.0.1:1080;
    export NO_PROXY=socks://127.0.0.1:1080;"
alias unproxy="
    unset HTTP_PROXY;
    unset HTTPS_PROXY;
    unset ALL_PROXY;
    unset NO_PROXY;"

使用 .zshrc

source ~/.zshrc

测试 proxy 效果

curl ip.sb
>> xxx.xxx.xxx.xxx
